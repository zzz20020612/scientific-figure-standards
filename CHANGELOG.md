# Changelog

All notable changes to this project will be documented in this file.

## [0.4.0] - 2026-05-22

### Added

- `references/submission-standards.md` for final manuscript, journal, thesis, and publication-ready figure rules.

### Changed

- Submission outputs now default to PNG 300 dpi, TIFF 300 dpi, and PDF.
- SVG is no longer generated by default; it is reserved for explicit user requests or journal requirements.
- Multi-panel submission labels now use `a`, `b`, `c` without parentheses.
- Submission typography now preserves the user's existing code/template font sizes by default.
- Submission rules now explicitly avoid extra metadata/notes files and keep long scientific explanations out of the figure body.

## [0.3.0] - 2026-05-22

### Changed

- Upgraded `no_text` behavior from text deletion to transparent text placeholders that preserve layout.
- Updated all bundled templates to create text-bearing objects first, then hide text for no-text outputs.
- Made world-map left/bottom graticule labels a hard standard, including the user's original `xlocs=[-90, 0, 90]`, `ylocs=[90, 60, 30, 0, -30, -60, -90]`, unrotated labels, and size-10 styling.
- Updated regional map guidance to use sparse, concise left/bottom graticules when graticules are needed.
- Updated colorbar no-text handling to preserve tick positions and label space instead of deleting ticks or labels.

## [0.2.0] - 2026-05-20

### Added

- Template-first replication layer for closer house-style reproduction.
- `references/template-guide.md` describing when and how to use templates.
- `templates/scatter_regression.py` for observed-vs-predicted train/test model plots.
- `templates/scenario_bar.py` for scenario reduction-efficiency bars.
- `templates/cumulative_distribution.py` for ranked cumulative contribution curves.
- `templates/broken_axis_bar.py` for skewed contribution bar charts with visible broken axes.
- `templates/shap_swarm.py` for SHAP swarm plots with mean absolute SHAP bars.
- `templates/global_raster_map.py` for global GeoTIFF/raster maps.

### Changed

- `SKILL.md` now requires using templates first when a requested figure matches an existing pattern.
- Visual and map standards now point agents to templates for exact replication.

## [0.1.0] - 2026-05-20

### Added

- Initial portable AI-agent skill for scientific figure and map standards.
- Codex / Claude Code entrypoint via `SKILL.md`.
- OpenCode and generic agent entrypoint via `AGENTS.md`.
- Claude Code fallback project instruction file via `CLAUDE.md`.
- Visual standards reference for typography, palettes, axes, legends, exports, and QA.
- Map standards reference for world/regional maps, CRS handling, GeoTIFF/raster cleaning, colorbars, and global map ramps.
- Data standards reference for unit conversion, missing data, model metrics, statistics, and output naming.
- Anonymized current-project style reference with extracted house-style examples.
- Python helper scripts for figure and map styling.
- MIT License.

### Notes

- North-arrow and scale-bar standards are intentionally not defined in this first version. They are reserved for a later update.
